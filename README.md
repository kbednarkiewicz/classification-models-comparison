# Classification Models Comparison

This project focuses on supervised learning methods for classification problems.

The goal was to build and compare different classification models, evaluate their performance, and analyze factors influencing classification accuracy.  
The project was completed as part of a university course. The report was written in Polish.

## Authors

- Katarzyna Bednarkiewicz
- Ewelina Dąbrowa
  
## Datasets
- **Iris Dataset** - used for building a regression-based classification model using binary class indicators.
- **Glass Dataset** – used for comparison of different classification algorithms.

## Analysis

The project included:

- data preparation and exploratory analysis,
- construction of a linear regression model using binary class indicators,
- polynomial feature expansion,
- comparison of classification algorithms:
  - k-Nearest Neighbors (k-NN),
  - classification trees,
  - Naive Bayes classifier,
- evaluation using confusion matrices and classification errors,
- comparison of models using different feature subsets, parameter selection, and validation approaches (bootstrap and cross-validation),
- interpretation of results and identification of the best-performing methods.
